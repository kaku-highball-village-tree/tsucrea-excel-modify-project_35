# 粗利ランキング差し込み仕様（PJ別側準拠・再表示）

## 前提
- 「粗利ランキング側を、PJ別（売上・売上原価・販管費・利益率）側の差し込み仕様に寄せる」方針。
- 本ドキュメントは**実装変更なし**で、変更後イメージを再表示したもの。

## 対象
- テンプレート: `TEMPLATE_PJサマリ_単月・累計_粗利金額ランキング.xlsx`
- 差し込み処理の方式を `TEMPLATE_PJサマリ_PJ別_売上・売上原価・販管費・利益率.xlsx` 側に合わせる。

## 1. 入力
- 入力は現状どおり粗利ランキング用TSVを使用。
  - `0002_PJサマリ_step0010_単月・累計_粗利金額ランキング.tsv`
- PJ別側は複数TSV探索だが、粗利ランキング側は現状の単一固定入力を維持。

## 2. シート運用
- 先頭シートを使って書き込む（必要に応じてシート名設定）。
- PJ別側のような複数シート展開は、入力性質が異なるため必須ではない。

## 3. 値書き込み
- `parse_tsv_value_for_excel` を通してA1起点で順次書き込み。
- この点はPJ別側と共通。

## 4. 書式ルール（PJ別側に合わせるポイント）
- 2行目以降は `number_format` のみ、テンプレ基準行（2行目、なければ1行目）からコピー。
- `alignment` はコピーしない（= PJ別側の挙動）。

## 5. 追加変換（粗利ランキング固有ロジックの扱い）
- `＋∞/－∞` 列検知による利益率 `%` 文字列化ロジックは適用しない。
- 目的はPJ別側仕様へ寄せること。

## 6. 出力
- 出力先・ファイル名は現状の粗利ランキング側を維持。
  - `PJサマリ/PJサマリ_単月・累計_粗利金額ランキング.xlsx`
- 必要なら実行ルート配下へのコピーも現状踏襲。

## 差分要約（現状 → ご希望仕様）
- 現状粗利ランキング側: `alignment` コピー + `%` 文字列化あり。
- ご希望（PJ別寄せ）: `number_format` コピーのみ、`alignment` コピーなし、`%` 文字列化なし（= PJ別流儀）。
